# C10---Projet-de-data-science
Projet final C10 : Capstone en science des données appliquée

Ce projet de Data Science a pour objectif d'analyser les missions Falcon 9 de SpaceX et de prédire le succès de l'atterrissage du premier étage de la fusée.

Le projet couvre l'ensemble du processus de Data Science :
Collecte des données (API et Web Scraping)
Nettoyage et préparation des données
Analyse exploratoire (EDA)
Visualisation des résultats
Machine Learning et prédiction

Utilisation des outils : Python, Pandas, NumPy, SQL / SQLite, Matplotlib, Plotly Dash, Scikit-Learn.

Principaux résultats :
Amélioration progressive du taux de réussite des atterrissages Falcon 9, identification des facteurs influençant le succès des missions et construction de plusieurs modèles de classification.

Structure du projet :

Ex1 : Data Collection via API

Ex2 : Web Scraping

Ex3 : SQL Analysis

Ex4 : Exploratory Data Analysis (EDA)

Ex5 : Interactive Maps with Folium

Ex6 : Interactive Dashboard with Dash

Ex7 : Machine Learning Prediction

Auteur A.M.

Projet réalisé dans le cadre d'une formation en Data Science (C10 : Capstone en science des données appliquée).
